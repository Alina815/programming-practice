# programming-practice
Предметная область: автоматическая телефонная станция. В классе хранятся два списка тарифов - обычные и льготные. В главном окне по нажатию соответствующих кнопок реализованы функции добавления обычного тарифа, добавления льготного тарифа, подсчёта средней цены (для льготных тарифов берётся стоимость с учётом скидки). По нажатию кнопок "Показать список тарифов" и "Показать историю операций" открывается новое окно (одна и та же форма с перегрузкой конструктора), отображающая нужную информацию. Список тарифов загружается из базы данных "Tariff.db" (в режиме по умолчанию) или из файла "tariffs.txt" (в локальном режиме). Отображается в виде таблицы с помощью DataGridView. В окне просмотра списка тарифов также доступны загрузка списка тарифов из файла, выбираемого с помощью OpenFileDialog, сохранение в файл; а также сортировки тарифов по цене, городу 1 и городу 2. История по умолчанию загружается из файла "history.txt" перед началом работы и сохраняется в него после её окончания. Во всей работе реализованы всяческие проверки ввода (пустые значения, переполнения int и double, строки вместо чисел, формат строк в файлах и тому подобное).
