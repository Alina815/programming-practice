# lab4
Предметная область: автоматическая телефонная станция. В классе хранятся два списка тарифов - обычные и льготные. В главном окне по нажатию соответствующих кнопок реализованы функции добавления обычного тарифа, добавления льготного тарифа, подсчёта средней цены (для льготных тарифов берётся стоимость с учётом скидки). По нажатию кнопок "Показать список тарифов" и "Показать историю операций" открывается новое окно (одна и та же форма с перегрузкой конструктора), отображающая нужную информацию. Список тарифов реализован с помощью DataGridView. В режиме просмотра списка тарифов также доступны загрузка списка тарифов из файла, выбираемого с помощью OpenFileDialog, и сортировки тарифов по цене, городу 1 и городу 2. Список тарифов и история по умолчанию загружаются из файлов "tariffs.txt" и "history.txt" соответственно перед началом работы и сохраняются в эти файлы после её окончания. Во всей работе реализованы всяческие проверки ввода (пустые значения, переполнения int и double, строки вместо чисел, формат строк в файлах и тому подобное).
